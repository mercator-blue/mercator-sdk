Notes

OpenLayers binding: raster decoded declaratively via WebGLTileLayer style expressions (no GLSL); all three encodings supported (scalar, vector magnitude, mapbox_rgb elevation), log scale via pre-curved palette. NEAREST sampling for byte-exact 16-bit decode. Overlays: arrows, value-labels, streamlines (WebGL2 + FBO trails), contours (Canvas2D MVT + along-line labels), tile-boundaries. v1 gaps on raster: smooth (in-value bilinear) and continuous coverage-alpha are not wired (validity is a hard cutoff).
